## 1. 시스템 개요

| 항목 | 내용 |
|------|------|
| 프린터 | Formlabs Form 4 × 4대 |
| 백엔드 | FastAPI (Python 3.11) |
| 프론트엔드 | React 18 + TypeScript + Tailwind CSS 4 |
| 데이터 | SQLite (프리셋/작업), Formlabs Cloud API (모니터링) |
| 서버 | 6000 서버 (개발), 카카오 클라우드 (운영 예정) |
| API | Web API 11개 + Local API 50개 = **총 61개 엔드포인트** |

### 연결 구조

```
브라우저 → http://106.244.6.242:8085/
    ↓
6000 서버 (FastAPI)
    ├── Formlabs Cloud API (모니터링, 15초 폴링)
    └── WireGuard VPN → 공장 PC
         ├── PreFormServer :44388 (슬라이스/프린트 전송)
         ├── file_receiver :8089 (STL 파일 수신)
         └── Form 4 × 4대 (WiFi)
```

## 2. 탭별 기능 상세

### 2-1. 모니터링 탭

프린터 4대의 실시간 상태를 한눈에 확인하는 메인 화면.

**프린터 카드 (4대 그리드)**
- 상태 표시: 출력 중(파랑) / 예열(주황) / 일시정지(노랑) / 중단(빨강) / 대기(초록) / 오류(빨강) / 오프라인(회색)
- 현재 작업명 + 진행률(%) + 진행 바
- 레이어 정보 (현재/전체)
- 프린트 단계 + 온도 (예: "Preprinting 28.5°C")
- 경과 시간 / 남은 시간 / 전체 예상 시간
- 레진 잔량 바 + 퍼센트 + 부족 경고
- 카트리지/탱크 미장착 경고

**상태 필터**
- 상단 요약 카드 클릭: 전체 / 출력 중 / 대기 / 오류·오프라인 필터링

**타임라인 간트 차트**
- 24시간 시간축, 프린터별 작업 바 (색상=상태)
- 날짜 이동 (이전/다음) + 미니 달력 (작업 있는 날 점 표시)
- 작업 바 호버 시 상세 정보 표시

**프린터 상세 모달**
- 프린터 이름 클릭 → 슬라이드 패널 (3탭: Details / Settings / Services)
- 기기 정보, 펌웨어, 카트리지/탱크 상세, 빌드 플레이트 상태

### 2-2. 프린트 제어 탭

프린터별 독립 컨테이너로 STL 업로드 → 슬라이스 미리보기 → 프린트 전송.

**프린터별 독립 워크플로우**
- 상단 프린터 탭으로 빠른 이동 (상태 점 표시)
- 각 프린터마다 개별 파일 업로드 + 프리셋 + 설정

**프린트 준비 체크 (Readiness)**
- 프린터 대기 상태일 때 6가지 자동 검증:
  1. 프린터 준비 상태 (READY)
  2. 레진 카트리지 장착 여부
  3. 레진 탱크 장착 여부
  4. 빌드 플레이트 설치 여부
  5. 빌드 플레이트 비어있는지 (이전 부품 확인)
  6. 프린터 온라인 여부
- 미충족 시 노란색 경고 배너 + 프린트 버튼 비활성화

**STL 파일 업로드**
- 드래그앤드롭 또는 파일 선택 (100MB 제한)
- 공장 PC의 file_receiver를 통해 전송

**프리셋 관리**
- 프린터별 독립 프리셋 (각 프린터에 개별 저장)
- 프리셋 항목: 이름, 부품 타입, 레진 종류, 레이어 두께, 서포트 밀도, STL 파일
- 생성 / 삭제 / 선택 적용
- 사용 횟수 자동 카운트

**슬라이스 미리보기**
- PreFormServer에 Scene 생성 → 슬라이스 실행
- 결과: 예상 시간 (초 단위 정밀), 예상 재료(ml), 3D 스크린샷
- 유효성 검사: 서포트/빌드 영역 사전 검증 (통과/경고/에러)

**고급 옵션**
- 내부 비우기 (Hollow): 벽 두께 설정, 레진 절약
- 모델 복제: 1~10개, 자동 재배치 + 간섭 검사

**프린트 전송**
- 모든 조건 충족 시 프린트 버튼 활성화
- 전송 완료 후 초기화

### 2-3. 대기 중인 작업 탭

프린트 대기열 관리.

- 프린터별 필터링 (전체 / 개별 프린터)
- 작업 상태: 대기(pending) / 준비 중(preparing) / 준비 완료(ready) / 전송 중(sending)
- 드래그앤드롭 순서 변경
- 예약 출력 시간 표시 (KST)
- 30초 자동 새로고침
- 작업 삭제

---

### 2-4. 이전 작업 내용 탭

과거 프린트 이력 조회 + 재출력 + 메모.

**이력 소스**
- 로컬 이력: 이 시스템에서 전송한 작업
- 클라우드 이력: Formlabs Cloud API에서 가져온 전체 이력

**필터링**
- 기간: 직접 입력 또는 빠른 선택 (1일 / 1주 / 1개월 / 3개월)
- 결과: 전체 / 성공 / 실패 / 중단
- 프린터: 전체 또는 개별

**작업별 표시 정보**
- 상태 (색상 뱃지), 작업명, 프린터명
- 시작 시간, 소요 시간
- 레진 종류 + 사용량(ml)
- 레이어 수
- 오류 메시지 (실패 시)

**재출력 기능**
- 프린터 선택 (레진 종류/잔량 표시)
- 바로 출력 또는 대기열 예약
- STL 파일 / 레진 / 레이어 두께 변경 가능

**메모 기능**
- 작업별 메모 추가/수정/삭제
- 인라인 표시

**CSV 내보내기**
- 기간 + 결과 필터 → CSV 다운로드
- 항목: 작업명, 프린터, 상태, 시작 시간, 소요 시간, 레진, 사용량, 레이어 수

### 2-5. 통계 탭

재료 사용량, 출력 추이, 프린터 가동률 시각화.

**필터**
- 프린터: 전체 또는 개별
- 기간: 직접 입력 또는 빠른 선택

**차트 4종**

| 차트 | 유형 | 내용 |
|------|------|------|
| 재료 사용량 | 도넛 차트 | 레진별 비율 + 사용량(ml) 테이블 |
| 일별 출력 수 | 바 차트 | 날짜별 출력 건수 |
| 프린터별 가동률 | 멀티 바 차트 | 프린터별 대기/출력/오류/오프라인 비율(%) |
| 프린터별 재료 | 수평 누적 바 | 프린터별 레진 종류별 사용량 |

### 2-6. 알림 시스템

**헤더 벨 아이콘**
- 미읽음 알림 수 뱃지 (빨간색)
- 클릭 시 드롭다운 패널

**알림 종류**

| 이벤트 | 아이콘 | 색상 |
|--------|--------|------|
| 출력 완료 | ✓ | 초록 |
| 출력 오류 | ! | 빨강 |
| 레진 부족 | ▼ | 노랑 |
| 프린터 오프라인 | ○ | 회색 |
| 출력 시작 | • | 파랑 |

- 30초 자동 갱신
- 전체 읽음 처리 버튼
- 상대 시간 표시 ("방금", "5분 전", "2시간 전")

### 2-7. 자동화 탭 (한솔코에버 개발)

프리셋 기반 작업지시 → 프린터 출력 → 로봇 후처리 자동 공정.

**현재 동작 흐름**
1. 프리셋으로 생성한 파일로 **작업지시** 등록
2. **작업 시작** → 작업지시를 통해 프린터 출력 시작
3. 출력 완료 → 프린터 상태에 따라 **큐에 저장된 순서대로** 로봇이 제품 이동
4. 로봇이 세척 → 건조까지 자동 진행
5. 정해진 시나리오에 맞춰 **로봇과 Modbus TCP 통신**으로 시퀀스 동작

**로봇 통신**
- 프로토콜: Modbus TCP
- 제어 대상: HCR-12 (빌드플레이트 교체, 세척기 투입), HCR-10L (후가공, 이송)
- 시퀀스 서비스가 로봇-프린터 간 작업 흐름을 오케스트레이션

**시퀀스 종류**
- 프린트 디스패치 (print_dispatch): 프린터 → 빌드플레이트 픽업
- 후처리 (post_process): 세척 → 경화 → 건조

---

### 2-8. 자동화 수동제어 탭 (한솔코에버 개발)

로봇-PC 간 I/O를 수동으로 테스트할 수 있는 디버깅/점검 도구.

**기능**
- 정해진 Modbus TCP 주소를 통해 로봇과 제어 PC 간 **I/O 테스트**
- 개별 입출력 포트 읽기/쓰기
- Ajin IO 보드 제어 (AXL.dll 기반)
- 시뮬레이션 모드 지원 (로봇 미연결 시)

## 4. 16단계 공정 흐름에서의 웹 대시보드 역할

```
① STL 업로드 (웹)              → 프린트 제어 탭          ✅
② 슬라이스 + 프린터 전송        → 프린트 제어 탭          ✅
③ 3D 프린팅                    → 모니터링 탭 (실시간)     ✅
④ 프린팅 완료 감지              → 알림 시스템             ✅
⑤~⑫ 로봇 작업 (세척/경화/후가공) → 자동화 탭 (한솔코에버)  별도 정리
⑬ 비전 검사                    → Phase 5 (개발 중)
⑭~⑮ 분류/적재                  → 자동화 탭 (한솔코에버)  별도 정리
⑯ 완료 보고                    → 알림 시스템             ✅
```

## 3. Phase 5 빈피킹 현황 (카메라 입고 전 SW)

### 3.1 파이프라인 완성도

| 단계 | 모듈 | 상태 | 성능 |
|------|------|------|------|
| L1 영상취득 | `realsense_capture.py`, `basler_capture.py`, `depth_to_pointcloud.py` | ✅ 완성 | D435 라이브 검증 + Basler 시뮬 PASS |
| L2 전처리 | `cloud_filter.py` (ROI→SOR→voxel→RANSAC→법선) | ✅ 완성 | 0.29s @ 186K pts |
| L3 분할 | `dbscan_segmenter.py` | ✅ 완성 | 과분할 <5%, 미분할 <3% |
| L4 인식+자세 | `pose_estimator.py` (FPFH+RANSAC/FGR+ICP+Colored ICP) | ✅ 완성 | easy 100%, crowded 90%, hard 60% |
| L5 그래스프 | `grasp_planner.py`, `grasp_database.yaml` (29종) | ✅ 완성 | validate_pick 안전 검증 |
| L6 로봇 통신 | `modbus_server.py` (HCR-10L INT16) | ✅ 완성 | Reg 130~151 / 400~405 / 600 / 700~702 |

### 3.2 성능 지표 (합성 씬 + D435 실데이터)

| 지표 | 목표 | 현재 | 상태 |
|------|------|------|------|
| 부품당 매칭 시간 | 2.0s | **0.4~0.6s** | ✅ 3~5배 여유 |
| 인식률 (easy) | 85% | **100%** (5/5) | ✅ |
| 인식률 (crowded 10종) | 80% | **90%** (9/10) | ✅ |
| 인식률 (hard ±45°) | 85% | 60% (3/5) | ⚠️ FPFH 한계 → Colored ICP 대기 |

| D435 Full Pipeline | 오탐 없음 | ACCEPT 0 (2회 모두) | ✅ RMSE 3mm 안전장치 확인 |
